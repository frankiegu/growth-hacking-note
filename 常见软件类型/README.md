# 常见软件类型
* 日常管理
  * OA： 考勤，请假，报销审批。
  * HRM：员工管理（考勤，绩效 =>工资），请假，报销审批流程。
  * PMS（资产管理）：酒店用的比较多。
  * 财务软件
* 产品生产管理系统
  * ERP： 进销存，供应链管理，财务结算。
  * MES
  * SRM（供应商关系管理） 供应商，库存等。
* 产品生产后的销售系统
  * 2B
    * CRM：客访管理（销售线索/机会/潜在客户，公海客户，私海客户，客户跟进），订单管理（合同管理，回款管理），报销，销售的培训，工具（群发短信，群发邮件），渠道，，客户投诉，竞争对手，销售预测(目标达成系统)，转化率漏斗，各种报表，BI。
  * 2C
    * 商城：商品管理，订单管理，促销管理（秒杀，团购，砍价，优惠），会员（等级，积分规则，打标签），财务（订单流水），其他（客户投诉）。
* 其他
  * CMS
