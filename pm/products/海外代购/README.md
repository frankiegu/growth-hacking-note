# 海外代购
## 代购的原因
艾瑞咨询2015年中国跨境网络购物用户研究报告中指出，2014年中国跨境网络购物用户选择跨境网购主要有以下几点原因：
* 产品品质有保证。 67.8%。
* 产品价格便宜。 65.5%。
* 产品品牌偏好。 53.0%。
* 产品稀缺：国内网站买不到。 52.0%。
* 产品丰富：可供选择多。 46.7%。
* 出国买过国外产品，有继续网购的意愿。 39.6%。
